# Micropile ASD Design Calculator

General preliminary micropile axial design/check web app.

## Main features

- Cased, uncased, bar-only, hollow-bar, and casing-only axial checks.
- Solid threaded bars, hollow/core bars, and custom bars from a generic bar database.
- Steel pipe casing sizes from a generic casing database.
- Multiple reinforcement groups, including partial-length bars.
- Multilayer grout-to-ground bond capacity below the casing end.
- Schematic pile/soil profile drawing.
- DRAFT PDF calculation report export with full calculations and no company branding.

## Deploy on Streamlit Community Cloud

Repository: your GitHub repository

## Notes

This app provides preliminary calculations only. Final design must be reviewed and sealed by the Engineer of Record, including lateral loading, bending, buckling, eccentricity, corrosion, couplers, group effects, load testing, and pile-cap connection design.
